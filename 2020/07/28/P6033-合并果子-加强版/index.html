P6033 合并果子 加强版

2020-07-28
OI

合并果子的加强版，扩大了数据范围

  • Subtask 1（10 points）：$1 \leq n \leq 8$。
  • Subtask 2（20 points）：$1 \leq n \leq 10^3$
  • Subtask 3（30 points）：$1 \leq n \leq 10^5$
  • Subtask 4（40 points）：$1 \leq n \leq 10^7$

对于全部的测试点，保证$1 \leq a_i \leq 10^5。$

由于n太大，复杂度必须为O(N)

发现a[i]不大，可以使用桶排

不能使用优先队列，改用数组模拟
